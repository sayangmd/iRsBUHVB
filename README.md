# 项目简介

## 核心功能
- 知识产权代管理系统
- 协同过滤算法
- 在线咨询
- Echarts可视化统计
- 系统首页
- 用户管理
- 知识产权人管理
- 产权类型管理
- 知识产权管理
- 知识产权变更
- 知识产权缴费
- 在线投诉
- 系统管理

## 适用场景
- 知识产权管理
- 数据分析
- 在线咨询
- 系统集成

## 技术特点
- 架构：B/S、MVC、Web框架
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
- 技术栈：Java、SpringBoot、Vue
